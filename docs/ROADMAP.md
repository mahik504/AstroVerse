# AstroVerse Roadmap

AstroVerse is currently in the **Research Execution Phase**. The core engineering and pipeline architecture is frozen while empirical benchmarking is underway.

## Status Checklist

- [x] **Engineering Freeze**: Architecture, API, and UI are fully implemented and stable.
- [x] **Documentation**: API, Architecture, and Reproducibility guides are complete.
- [x] **Reproducibility**: Zenodo archival scripts, environment manifests, and automated `Makefile` execution are in place.
- [ ] **Large Dataset**: Finalize generation of `v4-paper-10000` via automated MAST scraping.
- [ ] **Baselines**: Execute `evaluate.py` across 5 random seeds on the 10K dataset.
- [ ] **Paper**: Finalize `paper/main.tex` with robust statistical metrics and analysis.
- [ ] **Publication**: Submit to open-access preprint server (e.g., arXiv) and release v1.0.0.
